## Sources

Download official logos from:
- Wikipedia Commons (https://commons.wikimedia.org/)
- Official bank brand portals
- Brandfetch (https://brandfetch.com/)

## Specifications

- Format: SVG (preferred) or PNG with transparent background
- Max dimensions: Height should be 36px for consistent display
- File size: Under 150 KB per logo
- Optimization: Use SVGO or similar tools to optimize SVG files

## Usage

These logos are displayed in:
1. Trusted Partners section (header)
2. Card recommendation cards (alongside card images)

Make sure to use official, licensed logos and comply with brand guidelines.
